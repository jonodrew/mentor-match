GOV.UK Notify message templates

The matching process creates four possible outcomes. These outcomes are:

  1. mentees receive at least 1 match
  2. mentees receive no matches
  3. mentors receive at least 1 match
  4. mentors receive no matches

In GOV.UK Notify, you should have an email message template set up that explains the outcome of the matching process to mentors and mentees in each of these scenarios.

Make sure each template contains the following template fields:

  1. (( first name ))
  2. (( last name ))

For the templates to mentors and mentees with matches, also include these template fields:

  1. (( number of matches ))
  2. (( match details ))

Enter the message template IDs for each template in the fields below.
